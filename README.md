🚀 Cadastro de Usuários API

API REST desenvolvida com **Spring Boot** para gerenciamento de usuários, permitindo operações completas de CRUD (Create, Read, Update, Delete).

---

## 📌 Funcionalidades

- ✅ Criar usuário  
- ✅ Listar usuários  
- ✅ Atualizar usuário  
- ✅ Deletar usuário  

---

## 🛠️ Tecnologias utilizadas

- Java 17+
- Spring Boot
- Spring Data JPA
- H2 Database (em memória)
- Maven
